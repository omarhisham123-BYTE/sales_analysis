## 📌 Project Description
An End-to-End Pizza Sales Analysis project. Leveraging SQL Server for data querying and KPI validation, and Power BI for interactive storytelling. This project features deep-dives into sales trends, product performance, and customer behavior to drive data-driven business decisions.

## 🛠 Tools Used
- **SQL Server:** Data extraction & KPI calculation.
- **Power BI:** Data visualization & Dashboarding.

## 📁 Files in Repository
- `SQLQuery pizza_sales.sql`: SQL scripts for analysis.
- `pizza sales.pbix`: Power BI dashboard file.
